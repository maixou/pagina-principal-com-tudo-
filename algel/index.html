O que é HTML?

HTML (HyperText Markup Language) é a linguagem fundamental para criar páginas web.

Significado de cada letra:

  • H - HyperText (Hipertexto)
  • T - Text (Texto)
  • M - Markup (Marcação)
  • L - Language (Linguagem)
